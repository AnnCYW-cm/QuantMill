    研究纪要:横截面选股调查 / Research Notes¶

    一段完整的、诚实的量化研究闭环:从"发现惊艳回测"到"把它层层拆穿"。 这份记录本身就是 quantmill 最想传达的东西 —— 量化里营销 >> 真实收益,工具的价值是帮你分辨真伪,而不是让你上头。 数据:CSI300(A股)+ 港股蓝筹,2023-01 起,horizon=20 天,月度换仓,含 0.15% 单边成本。

    0. 一句话结论¶

    复杂 ML 排名模型是过拟合幻觉;简单价值/动量/低波因子组合是"真但弱"的边际。 四道独立关卡(可信度体检 → 子区间衰减 → 跨市场验证 → 前视偏差量化)一致指向同一结论。

    1. 范式:从"单股预测"到"横截面选股"¶

    对标 Qlib/Numerai 发现的根本缺口:平台原本给每只股票各训一个模型预测它自己涨跌(时序),这学不到"今天买 A 还是买 B"。真正的选股是横截面:全市场一个模型,每天给所有股票打分排名,买最强、(可)卖最弱。

    • 时序 IC:一只票,因子随时间 vs 它自己未来收益(纵向)。
    • 横截面 IC:同一天,所有票的因子 vs 各自未来收益(横向排名),按天平均。选股依赖后者。

    据此建了 cross/ 模块(universe → panel → ic → model/composite → backtest)。

    2. 横截面 IC(全 CSI300,300 只)¶

    因子 IC ICIR 读法
    ey (=1/PE) +0.048 0.18 盈利便宜跑赢 —— 价值最强
    size (ln市值) −0.033 −0.18 小盘跑赢 —— 小盘效应
    dist_high_20 +0.030 0.14 近高点继续强 —— 动量
    bp (=1/PB) +0.030 0.12 账面便宜跑赢
    vol_10/20/5d −0.02~−0.03 低波动异象

    教训(样本量骗人):10 只小样本里 ret_60d 动量 IC +0.081、bp −0.13;扩到 300 只后 ret_60d 翻负、bp 反号(那 −0.13 全是万科一只带偏)。能扛过样本放大的,才是真因子。

    3. ML 排名模型:先惊艳,后拆穿¶

    purged walk-forward + LightGBM,43 因子。初见惊艳:

    总收益 年化 夏普 超额年化
    ML top-20 +82.4% 56.1% 1.58 +33.5%
    等权基准 +31.7% 22.6% 1.17 —

    3.1 可信度拷问¶

    检验 结果 判读
    k 敏感性 k5:+108% → k100:+7.7%(单调) 有结构,但集中度赌博
    成本敏感 扣到 1% 单边仍 +18% 不是换手幻觉
    子区间 前半 +4.42%/期(88%胜)→ 后半 +0.22%/期(56%胜) 🔴 近9月 alpha 蒸发
    size tilt top-20 平均 size 分位 0.30 🔴 系统性押小盘
    DSR 0.90(样本外仅17期) ❌ 没过 0.95
    PBO 0.41(T小仅参考) ⚠️ 略好于抛硬币

    初步定性:这不是 AI 选股 alpha,是小盘因子择时 —— 2025 微盘行情吃出来的,行情一冷就现原形。

    3.2 决胜:size 中性化(证伪"纯小盘")¶

    在市值分档内选股(锁死市值构成),超额: - 原始 +33.5% → 同模型 size 中性 +21.1% → 最严(删 size 特征 + 标签残差化)+49.5%。 - 证伪了"纯小盘赌注":去掉市值后仍有档内选股力,甚至更强。

    但 size 中性的子区间衰减:504 窗(纯2025,17期)+2.79%/期未衰减;放到 252 窗(2024-2026,29期)缩到 +0.86%/期、胜率 55%、靠几个暴涨月撑着 —— 真,但弱,噪声主导。

    4. 跨市场验证:港股(最强的照妖镜)¶

    同一套方法搬到港股(86 只蓝筹,独立市场)。

    4.1 IC 同号复现¶

    因子 A股 IC 港股 IC 是否一致
    ey +0.048 +0.100 ✅ 同号更强
    低波动 vol_* −0.02~−0.03 −0.07~−0.08 ✅ 同号更强
    dist_high/动量 +0.03~ +0.05~+0.08 ✅ 同号
    bp +0.030 +0.062 ✅ 同号
    size −0.033(小盘赢) +0.054(大盘赢) ❌ 反号

    价值/低波/动量在两个独立市场同号复现 = 强证据它们是真普适 premia。size 反号 = 坐实 A股那点"小盘 tilt"是市场/regime 特有,不跨市场。

    4.2 决胜:ML vs 简单组合,跨市场¶

    A股 raw 港股 raw 跨市场
    43因子 ML 模型 +33.5% −10.2% ❌ 不复现(赢A输港)
    简单因子组合(零训练) +3.4% +15.4% ✅ 两地都正

    ML 换个市场直接亏钱(过拟合 A股结构);简单固定配方组合两个市场都赚,用满 42 期(非17期)。简单打败复杂。

    简单组合配方(跨市场同号因子,按符号加权分位):ey + bp + dist_high_20 + pos_60d − vol_20d,零训练。

    5. 前视/幸存者偏差量化(第三次坐实)¶

    用 index_stock_cons 的「纳入日期」发现:现有 300 只 CSI300 里 81 只(27%)是 2023 后才纳入的(涨得好才进来 = 前视污染)。取"2023 前已纳入"的 PIT 干净池(210 只) 重跑:

    模型 全300(含污染) PIT 210(干净) 偏差贡献
    稳健组合 +3.4% +12.5% −9.1%(去污染反而更好)
    ML 排名 +33.5% −7.7% 💥 +41.2%(去污染崩成负)

    ML 那 33.5% 里 41 个点是前视偏差;而稳健组合根本不依赖偏差,甚至受它拖累。 ⚠️ PIT 只修了"前视纳入"这一半;"起点在、后被踢出"的股票仍缺,需付费 PIT 数据。

    6. 最终判决与产品含义¶

    判决¶

    • 价值(ey/bp)+ 低波动 + 动量是真的 —— 两市场 IC 同号 + 简单组合都为正 + 去偏差后更稳。
    • 但边际很小 —— 档内超额 ~0.2–0.7%/期、胜率 52–55%,统计上勉强过半。真,但弱,不可重仓。
    • ML 是海市蜃楼 —— 港股翻车 + 前视偏差贡献 41 点,一换环境原形毕露。

    剥离过程(一张图看懂)¶

    阶段 表面数字 剥掉什么
    初见 ML 年化 56% / 超额 33.5% —
    剥 regime(子区间) 后半超额 +0.2% 前高后平
    剥小盘(size中性) 约 1/3 是纯押小盘 市值 tilt
    剥跨市场(港股) ML 港股 −10.2% 市场特异性
    剥前视偏差(PIT) ML +33.5% → −7.7% 幸存者/前视
    真身 单位数年化、55%胜率、真但弱 全部幻觉

    给产品的铁律¶

    1. 主推透明的稳健因子组合 + 可信度层,而非黑箱 ML。
    2. 能跨市场活下来的简单东西,胜过看着漂亮的复杂东西。
    3. 偏差要量化、当面标出来,不藏在漂亮回测后面。

    7. 复现命令 / Reproduce¶

    quantmill cross ic --market cn            # §2 横截面 IC
quantmill cross backtest --market cn --model ml         # §3 ML(会看到漂亮的假象)
quantmill cross backtest --market cn --model composite  # §4 稳健组合(DSR 0.989)
quantmill cross validate                  # §4 跨市场 A股+港股
quantmill cross survivorship --model ml         # §5 前视偏差:ML 崩成负
quantmill cross survivorship --model composite  # §5 前视偏差:组合反而更好

    ⚠️ 所有数字均带幸存者偏差(当前成分股)、单一 3.5 年样本、免费数据。结论方向可信,绝对幅度别当真。别拿真钱跟。
