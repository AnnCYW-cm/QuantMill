# 行为准则 / Code of Conduct

本项目采用 [Contributor Covenant](https://www.contributor-covenant.org) v2.1 行为准则。

## 我们的承诺

为营造开放、友好的环境,我们承诺:无论年龄、体型、身心障碍、族群、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向如何,参与本项目和社区的每个人都能获得免于骚扰的体验。

## 我们的准则

**鼓励:** 友善包容、尊重不同观点、优雅接受建设性批评、以社区利益为重、对他人展现同理。

**不可接受:** 性化言语或图像、人身攻击、公开或私下骚扰、未经许可公开他人隐私信息、以及其他在专业场合不当的行为。

## 执行

违规行为可发邮件报告至 **wootilehq@gmail.com**。所有投诉都会被审查与调查,并作出适当且必要的回应。维护者有权移除、编辑或拒绝不符合本准则的评论、提交、代码、issue 及其他贡献。
